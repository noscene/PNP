# PNP
drive a openplacer pick and place machine with small PyQt5 app


Install on MacOS
```
pip3 install pandas
pip3 install PyQt5
pip3 install serial
pip3 install matplotlib
pip3 install pyserial
pip3 install time
pip3 install numpy
pip3 install cv2
```

start with
```
python3 PNPApp.py
```
